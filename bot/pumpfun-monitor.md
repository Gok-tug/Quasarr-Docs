# Pump.Fun Monitor

Scans for significant events on Twitter, tracks tokens related to these events on Pump.fun and provides notifications about them along with the original tweet for reference.

## How It Works

1. The monitor scans Twitter for significant crypto related events
2. It identifies tokens created on Pump.fun that relate to these events
3. You receive a notification with token details and the original tweet

## What You Get

Each notification includes:
- Token information from Pump.fun
- Link to the original tweet
- Relevant metrics and data

## Use Case

This monitor helps you catch early opportunities when tokens are created in response to viral tweets or trending topics.
