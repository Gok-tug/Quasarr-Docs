# Wallets

Manage your blockchain wallets organized in groups.

## Groups

Groups help you organize wallets by project or purpose.

### Creating a Group

1. Click **New Group** in the left panel
2. Enter a name
3. Choose a color
4. Click Create

Each group shows wallet count and total balance.

### Managing Groups

- Click the pencil icon to edit name or color
- Click the trash icon to delete (removes all wallets in it)

## Creating Wallets

Generate new wallets directly in the app.

1. Select a wallet group
2. Click **New Wallet**
3. Enter the number of wallets to create
4. Optionally set an alias prefix
5. Click Create

Your new wallets are ready to receive funds.

## Importing Wallets

Bring existing wallets into Quasarr.

### Single Import
1. Select target group
2. Click **Import**
3. Enter the private key
4. Click Import

### Batch Import
Enter multiple private keys, one per line. You can also include aliases in the format: `alias,privatekey`

## Exporting Wallets

Backup your wallets for safekeeping.

1. Select wallets using checkboxes
2. Click **Export**
3. Choose save location
4. File saves with aliases and private keys

**Security Warning:** Export files contain private keys. Store securely.

## Balances

1. Select a network from the RPC dropdown
2. Click the refresh button
3. Balances update for all wallets

## Actions

Each wallet has quick action buttons:
- **Copy address** - Copy to clipboard
- **Copy private key** - Copy to clipboard
- **Delete** - Remove from app
