# Settings

Customize Quasarr to your preferences.

## Appearance

### Theme
Choose from available color themes. Options include light and dark variations.

### Language
Currently supports English. More languages coming soon.

## API Keys

### Moralis API Key

Required for NFT features.

1. Go to moralis.com and create a free account
2. Get your API key from the dashboard
3. Enter the key in Settings
4. Click **Save**

A green "Configured" badge appears when saved successfully.

## App Version

Your current version is shown at the bottom of Settings. The app checks for updates automatically at startup.
