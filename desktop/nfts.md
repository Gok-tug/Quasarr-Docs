# NFT Manager

View and transfer NFTs across your wallets.

## Setup Required

Before using NFT features, add your **Moralis API Key** in Settings. Get a free key at moralis.com.

## Viewing NFTs

### Load Your Collection

1. Select a wallet group from the left
2. Check which wallets to scan
3. Select the network RPC
4. Click **Load NFTs**

NFTs appear in the gallery.

### View Options

- **Card View** - Visual grid with images
- **List View** - Compact table format

### Filtering

- Search by name or collection
- Filter by blockchain
- Toggle **Hide Low Value** to clean up the view

### NFT Details

Click any NFT to see:
- Full image
- Token ID
- Collection name
- Owner wallet

## Transferring NFTs

### Single Transfer

1. Click on an NFT
2. Click **Transfer**
3. Select destination wallet
4. Confirm

### Batch Transfer

1. Select multiple NFTs using checkboxes
2. Click **Transfer**
3. Choose destination
4. Confirm

Each NFT transfer is a separate transaction.

## Tips

- Make sure source wallet has gas for transfers
- Some NFTs may have transfer restrictions
- Refresh the view after transfers to see updated ownership
