# Getting Started

Welcome to Quasarr Desktop documentation. This guide will help you set up and start using the application.

## Installation

1. Download Quasarr from the official website
2. Run the installer
3. Launch the application

## Login

Enter your license key to activate. Your license comes from:
- Holding a Quasarr NFT (Premium Access)
- Discord trial access for verified members

Check **Remember my license** to stay logged in on this device.

## First Time Setup

After login, follow these steps:

### 1. Add RPC Endpoints
Go to **RPC** from the sidebar. Add endpoints for the networks you want to use. This is required for all blockchain operations.

### 2. Create a Wallet Group
Go to **Wallets** and click **New Group**. Give it a name and color.

### 3. Add Wallets
Create new wallets or import existing ones into your group.

### 4. Ready to Go
You can now use Tasks, Transfers and NFT features.

## Dashboard

The dashboard shows:
- Total wallets and groups
- Recent task activity
- RPC connection status

## Navigation

Use the sidebar to switch between sections:
- **Dashboard** - Overview and stats
- **Wallets** - Manage wallets
- **Tasks** - Automation tasks
- **Transfers** - Send and receive funds
- **NFT** - View NFT collections
- **RPC** - Network connections
- **Settings** - App preferences
